# OOP2 – Domaći zadaci

Ovo je repozitorijum sa domaćim zadacima iz predmeta **OOP2**.

---

## 1. Prvi domaći zadatak – Torte 

**Opis:**  
Java aplikacija koja obrađuje podatke o ručno pravljenim tortama.  
- Implementira se `Torte.torteStream()`.  
- Zadatak uključuje: broj torti sa manje od pet preliva, lista sastojaka po torti, ukupna količina sastojaka, statistika po mesecima i obliku torti.  

**Folder sa kodom:** [`Prvi domaci zadatak/Prvi/src`](Prvi%20domaci%20zadatak/Prvi/src)

---

## 2. Drugi domaći zadatak – Kamen-Papir-Makaze-Guster-Spok

**Opis:**  
Java igra putem soketa između servera i klijenata.  
- Igra se dok neko ne osvoji tri poena.  
- Pravila igre:  
  - Kamen lomi makaze i gustera  
  - Papir pokriva kamen i Spoka  
  - Makaze seku papir i gustera  
  - Guster jede papir i Spoka  
  - Spok unistava kamen i makaze  

**Folder sa kodom:** [`Drugi domaci zadatak/Drugi/src`](Drugi%20domaci%20zadatak/Drugi/src)

---

## 3. Treći domaći zadatak – Kamen-Papir-Makaze-Guster-Spok preko RMI 

**Opis:**  
Distribuirana Java igra preko RMI.  
- Server registruje objekat za partiju, klijent udaljeno poziva metod `odigraj()`.  
- Igra traje dok neko ne skupi tri poena.  
- Pravila igre su ista kao u prethodnom zadatku.  

**Folder sa kodom:** [`Treci domaci zadatak/Treci/src`](Treci%20domaci%20zadatak/Treci/src)
